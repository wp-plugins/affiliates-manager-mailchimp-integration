An addon for the Affiliates Manager plugin to signup the affiliates to your MailChimp list

== Description ==

This addon allows you to specify a MailChimp list name in the addon settings. When affiliates join your site, they get signed up to the specified MailChimp list automatically.

This addon requires the [Affiliates Manager Plugin](https://wordpress.org/plugins/affiliates-manager/).

After you install this addon, go to the MailChimp settings interface and specify your API Key and the MailChimp list name.

Read the following page for step by step usage documentation:
http://wpaffiliatemanager.com/signup-affiliates-mailchimp-list/

== Installation ==

1. Upload `affiliatemgr-mailchimp-signup` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
